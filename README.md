# Joushi-no-mimi

## 概要
会社の愚痴や不平不満を『王様の耳はロバの耳』のようにうちあけるSNSアプリ。
ログイン・ログアウト機能、記事の投稿・検索、お気に入り記事へのイイネボタン、ニュースの閲覧・検索、プロフィールの編集ができます。

## 使用ライブラリ
*Firebase
*ESTabBarController
*SVProgressHUD
*CLImageEditor
*Kingfisher
